## Le Serveur

Lorsque l'on démarre le serveur il faut lui spécifier un port.
Si aucun port n'est spécifié, le port 8080 est utilisé par défaut.

## Le Client

Lorsque l'on démarre le client, il faut spécifier l'ip du serveur suivit de son port.
Si aucune ip n'est spécifié, localhost (127.0.0.1) est utilisé par défaut.
Si aucun port n'est pas spécifié, le port 8080 est utilisé par défaut.

## Le Jeu

Le jeu implémenté est une bataille où l'on peut voir ses cartes, chacun joue à tour de rôle.

## Les commandes

- HELP : affiche l'aide du jeu
- SCORE : affiche le score du joueur
- CARDS : affiche les cartes du joueur
- PLAY {card number} : joue la carte désigné
